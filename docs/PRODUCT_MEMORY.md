# Interview Mentor Product Memory

This app is for one learner opening it when focus is low. The first screen should make the next useful action obvious.

## Today Page Rule

- Show the next task card first.
- Keep labels short and human: Today, Start here, Notes, Time.
- Do not explain streak logic, carry-over logic, or behavior design on the main surface.
- Put strict mode, motivation tools, and psychology tricks behind a collapsed section.
- Remove planning/reflection forms from Today unless they directly help finish the current card.

## Card Rule

- A card needs a title, a plain description, minutes, and one action button.
- Avoid internal labels like system logic, carry-over, sprint theory, or roadmap mechanics.
- The user should understand what to do without reading instructions.

## Notes Rule

- Notes live on the Notes page, not inside Today.
- Preserve the raw note and add AI feedback separately.

## Discipline Rule

- Discipline tools should reduce friction to start, not create another task.
- Hard Mode is opt-in. It can add skip friction, but it should stay hidden until needed.
